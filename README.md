# 🚗 AquaWash — Demo Portfolio

> Sistema de gestión para lavadero de vehículos. Demo profesional construido con Next.js 15 + TypeScript + Tailwind CSS.

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/myesidpelaez/demo-lavadero)

## ✨ Features

- **Landing page completa** — Hero, Servicios, Cómo funciona, Precios, Testimonios, CTA
- **Reserva online en 4 pasos** — Wizard interactivo con validación
- **Documentación técnica** — Página `/docs` con stack, arquitectura y decisiones
- **Dark mode** — Toggle fluido sin flash
- **100% estático** — Deploy en cualquier CDN

## 🛠 Stack

| Tecnología | Versión | Rol |
|---|---|---|
| Next.js | 15.3 | Framework (App Router) |
| TypeScript | 5.x | Tipado estático |
| Tailwind CSS | 3.4 | Sistema de diseño |
| Lucide React | 0.511 | Iconografía |
| Vercel | — | Hosting & CI/CD |

## 🚀 Correr en local

```bash
git clone https://github.com/myesidpelaez/demo-lavadero
cd demo-lavadero
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000)

## 📦 Build

```bash
npm run build   # Genera /out con HTML estático
```
